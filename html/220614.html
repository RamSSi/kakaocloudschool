이미지 경로 1) 상대경로 : 보통 많이 사용하는 지정 방식 - '.' : 현재 디렉토리 기준 - '..' : 상위 디렉토리 기준 2) 절대경로 : 서버에는 많은 프로젝트의 이미지가 존재하기 때문에 절대경로 사용 시 수정이 어려움 웹에 사용하는 이미지는 파일 크기가 크지 않으면서 화질이 좋아야 함 1) jpeg, jpg : 비트맵 이미지(크기가 큼)를 압축한 것 2) gif 3) png : gif 유료화로 인해 개발된 무료 형식 오디오와 비디오 삽입 <object> 속성 : controls(컨트롤 바 유무), autoplay(자동재생여부), loop(반복재생여부), mute(소리 제거), width, height anchor(닻 = 링크) 속성 : name, target(문서를 띄울 방식) form 태그 안의 input이어야 서버로 데이터 전송이 가능 method 1) get(default) : 입력 길이 제한, 데이터 전송 시 사용자가 입력한 내용이 주소표시줄에 나타남, 한글 전송 X(별도의 가공 필요) 2) post(파일 전송 시 post 사용해야 함) : 입력 내용 길이 제한 X, 사용자가 입력한 내용 드러나지 않음 id와 name의 차이 id : 모든 태그들이 가질 수 있는 속성, 하나의 HTML 내에 유일한 요소의 이름 => 자바스크립트와 관련 name : 입력 양식 태그(input, textarea, select)만 가질 수 있는 속성, 중복 가능 => 서버와 관련 autocomplete : 자동완성, fieldset : 폼 요소 그룹, label : 폼 요소에 이름(라벨)을 붙임 input html 파일 간의 정보를 공유하기 위해서는 요소에 담아 전송해야 함 (html 문서 상에서 벌어진 이벤트를 다른 html 문서에서는 알 수 없음) - autofocus : 자동으로 입력 커서를 갖다 놓음 - required : 필수 작성 input - placeholder : 텍스트 입력란에 적당한 힌트 내용 표시
